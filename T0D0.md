# Culturepedia - Project Management

## Project Overview
Culturepedia is a clean, ownership-first web app that helps travelers and business people navigate cultural differences with practical, actionable hints.

Tech Stack: Static site (HTML + Tailwind + JavaScript), PWA support, deployed on Cloudflare Pages.

## Current Status
- Core functionality is working (filtering, search, upvote, submit, copy)
- Visual polish completed (June 2026)
- Region filter added (initial implementation)
- Deployed on Cloudflare Pages
